# 判断语句
JavaScript 中的 if-else 语句与 C++、Python、Java 中类似。

例如：


```javascript
let score = 90;
if (score >= 85) {
    console.log("A");
} else if (score >= 70) {
    console.log("B");
} else if (score >= 60) {
    console.log("C");
} else {
    console.log("D");
}
```
JavaScript 中的逻辑运算符也与 C++、Java 中类似：
- `&&`表示与
- `||`表示或
- `!`表示非
